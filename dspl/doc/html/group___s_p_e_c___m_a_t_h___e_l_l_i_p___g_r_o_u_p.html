Обратная эллиптическая функция Якоби вещественного аргумента

Функция рассчитывает значения обратной эллиптической функции для вещественного вектора w.
Для расчета используется итерационный алгоритм на основе преобразования Ландена.

Аргументы
[in]wУказатель на массив вектора переменной .
Размер вектора [n x 1].
Память должна быть выделена.

[in]nРазмер вектора w.
[in]kЗначение эллиптического модуля . Эллиптический модуль – вещественный параметр, принимающий значения от 0 до 1.

[out]uУказатель на вектор значений обратной эллиптической функции .
Размер вектора [n x 1].
Память должна быть выделена.

Возвращает
RES_OK Расчет произведен успешно.
В противном случае код ошибки.
Автор

int ellip_landen ( double  k,
int  n,
double *  y 
)

Расчет коэффициентов ряда полного эллиптического интеграла.

Полный эллиптический интеграл может быть представлен рядом:

где вычисляется итерационно при начальных условиях :

Данная функция рассчитывает ряд первых n значений , которые в дальнейшем могут быть использованы для расчета эллиптического интеграла и эллиптических функций.

Аргументы
[in]kЭллиптический модуль .
[in]nРазмер вектора y соответсвующих коэффициентам .

[out]yУказатель на вектор значений коэффициентов .
Размер вектора [n x 1].
Память должна быть выделена.

Возвращает
RES_OK Расчет произведен успешно.
В противном случае код ошибки.

Пример использования функции ellip_landen:

#include <stdio.h>
#include <stdlib.h>
#include "dspl.h"
#define N 14
int main()
{
void* handle; // DSPL handle
handle = dspl_load(); // Load DSPL function
double k[N];
int i, err;
err = ellip_landen(0.93, N, k);
if(err != RES_OK)
{
printf("Error code: %8x\n", err);
return err;
}
printf(" i%8sk[i]\n\n", "");
for(i = 1; i < N; i++)
printf("%2d %11.3e\n", i, k[i]);
dspl_free(handle); // free dspl handle
return 0;
}

Заметки
Ряд полного эллиптического интеграла сходится при значениях эллиптического модуля . При этом сходимость ряда достаточно быстрая и для практический приложений достаточно от 10 до 20 значений для обеспечения погрешности при расчете полного эллиптического интеграла в пределах машинной точности.
